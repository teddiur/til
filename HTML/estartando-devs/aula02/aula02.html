Resumo

HTTP

HTTP é um protocolo de comunicação na internet, que se estabelece um par requisição-resposta. Os principais métodos requisições são:

Método

Descrição

GET

Requisita informação do servidor.

POST

Envia informação pro servidor.

DELETE

... deleta.

PUT

Edita informação já no servidor.

Existe um padrão de respostas que pode ser acessado aqui. As respostas são divididas em faixas, como a seguir:

Tipo de resposta

Faixa de código

Respostas de informação

100-199

Respsotas de sucesso

200-299

Redirecionamentos

300-399

Erros do cliente

400-499

Erros do servidor

500-599

HTML

HTML foi criada em 1991 por Tim Berners-Lee no CERN sim, onde fica o colisor de partículas

É uma linguagem de marcação não de programação. Ela é lida pelo navegador que, então renderiza as páginas como as vemos.

Atividades

  • Criar pelo menos 04 páginas, linkadas entre si, utilizando apenas HTML.
