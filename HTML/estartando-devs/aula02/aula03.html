Tags semanticas HTML

HTML tem tags com significado como <nav> ou <header> que, além de servir como caixas (assim como <div>) estruturam o código.

Utiliza-las é uma boa prática para otimizar o site para o mecanismo de busca (Search Engine Optimization - SEO)
