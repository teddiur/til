Aula 04 - Seletores CSS e principais propriedades

Módulo1

Tópicos

  • Sintaxe CSS
  • Seletores CSS básicos
  • Principais propriedades

Resumo

O CSS atribuindo propriedades à elementos HTML que foram selecionados usando alguem tipo de seletor. A sintaxe é:

seletor {

propriedade: valor;

}

Os seletores básicos no CSS são as tags, as classe (declara-se class="name" dentro da tag e como seletor usa-se .name), os id (declara-se id="name" dentro da tag e é único, como seletor usa-se #name) e os atributos (pode-se utilizar qualquer atributo declarado numa tag com a sintaxe [atributo="valor"], há outros operadores além de =)

Atividades

  1. Estilizar a atividade 2 ou criar uma nova página para estilizar com CSS
