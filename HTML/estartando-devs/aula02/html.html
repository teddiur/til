HTML

HTML foi criada em 1991 no CERN por Tim Berners-Lee e é, até hoje, a linguagem base de toda a web. A versão mais atual é HTML5 desenvolvida pela W3C (Web World Wide Consortium). Veja um pouco da sua história:

A linguagem utiliza tags com sintaxe <tag> conteúdo </tag>. Se diz que o conjunto tag + conteúdo é um elemento.
Algumas tags não precisam ser fechadas (se diz que uma tag é fechada ao utilizar </tag>), alguns exemplos são img e video.
As tags são ditas filhas de outras quando estão completamente encapsulads pelas mães. Se diz que p é filha de body no exemplo a seguir:

<body>
<p>Algum texto que aparecerá no site</p>
</body>

As tags presentes em todo site são html, head e body. HTML é a tag mãe de qualquer elemento, enquanto head e body são irmãs. Dentro de head estão todas as meta-informações do site, enquanto body guarda o conteúdo.
Geralmente utiliza-se algum tipo de identificador para ser possível selecionar os elementos com alguma outra linguagem (CSS e JavaScript, por exemplo). Para CSS utiliza-se class (que não é um identificador único) e para JavaScript utiliza-se id (que recomenda-se ser único).
